# Daily Word Test

매일 날짜 기준으로 카테고리별 영어 단어 1문제를 보여주는 웹 앱입니다.

## 기능

- `GRE`, `TOEIC`, `수능` 카테고리 제공
- 카테고리별로 서울 시간 기준 매일 다른 문제 1개 표시
- 왼쪽에 단어와 발음기호 표시
- 오른쪽에 영영사전 스타일 4지선다 제공
- 정답 선택 후 맞고 틀림 결과와 정답 뜻 표시

## 실행

```bash
node server.js
```

또는

```bash
npm start
```

브라우저에서 `http://localhost:3000`으로 접속하면 됩니다.
